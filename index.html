IndexedDB POC

based on https://output.jsbin.com/gituxa

  • Creating a Database, for storing employees (an employee has an id (property "empId"), a name, a department and an email)
  • Creating an object store in that dataBase,
  • Specifying a keyPath (a property in the objects stored that will act as a Primary Key, in this case the empId),
  • Adding two indexes for faster retrievals (one on the name, one on the email (unique) property of each object),
  • Populating the dataBase with three entries.

Press button for calling createDatabase(). See debugging console.
